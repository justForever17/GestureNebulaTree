# GestureNebulaTree

单文件HTML手势控制星云圣诞树特效 | Gemini3.0pro 生成

## 🌟 项目介绍
一个无需依赖、仅需单个HTML文件即可运行的手势控制视觉特效项目。由 Gemini3.0pro 全程生成核心代码，通过摄像头识别手势，实现星云粒子在「凝聚态（圣诞树）」与「扩散态（钻石星辰）」之间的动态切换，支持插入多张自定义照片并随粒子同步移动，还能通过OK手势精准拉近展示照片，互动感拉满。

## 🎯 核心功能
- ✋ **手势交互核心**：
  - 握拳 → 粒子瞬间聚拢，形成带光泽的圣诞树形态
  - 张开手掌 → 7000颗钻石光泽星辰扩散，铺满全屏
  - 🤌（OK手势）→ 自动识别最近照片并拉近至屏幕中央，松开后复位
- 📸 **多照片支持**：可自定义插入多张照片，照片随星云粒子同步运动
- 🔄 **手势跟随旋转**：星云整体跟随手部位置实时旋转，互动更自然
- 📦 **单文件便携**：无任何外部依赖，下载后直接用浏览器打开即可运行

## 🚀 使用方法
1. 下载项目中的 `index.html` 文件
2. 用现代浏览器（Chrome/Edge/Firefox 等）打开该文件
3. 首次打开时允许浏览器访问摄像头（用于手势识别）
4. 开始互动：
   - 握拳 → 凝聚成圣诞树
   - 张开手掌 → 扩散为钻石星云
   - 比OK手势 → 拉近最近照片
   - 移动手部 → 星云跟随旋转

## 📸 自定义插入照片
1. 点击屏幕上的按钮
2. 选择本地照片，照片会实时入场
